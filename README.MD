# Sistema 9Dedos

Sistema de controle de estoque desenvolvido em Python + MySQL.

## Funcionalidades
- Login
- CRUD de produtos
- Controle de estoque
- Relatórios

## Tecnologias
- Python
- MySQL

## Como executar

obs.: database é local, por isso lembre de conferir os parametros de conexao.py

pip install mysql-connector-python

python main.py
